The elf package is software contributed by software elves, funded by lunch money. Elves, like hackers, write software for their own purpose, Luckily, sometimes they leave an editor buffer open that reveals what they've been up to. I've simply added some documentation and cleanup here and there.

I use a lot of this package every day. Studying these files, can help you learn JScheme and make effective use of Java API's.

  • src/elf/apropos.scm Apropos - find matching symbols.
  • src/elf/basic.scm Loads a standard set of files (prelude) that i use all the time.
  • src/elf/classpath.scm Classpath manipulation and dynamic class loading.
  • src/elf/describe.scm Describe an object or a class.
  • src/elf/future.scm Multilisp's future.
  • src/elf/GCMonitor.scm Loading this file brings up a simiple GC monitor pane.
  • src/elf/html-gen.scm Html generation.
  • src/elf/html.scm HTML document spider following Java Developer Tech Tip 923.
  • src/elf/inspect.scm A simple object inspector.
  • src/elf/interrupt.scm An interrupt button to interrupt runway computations.
  • src/elf/iterate.scm Defines (iterate) methods on sequences, collections anda arrays.
  • src/elf/jdbc.scm JDBC access.
  • src/elf/load.scm Probably should go away.
  • src/elf/mbe.scm Dorai Sitaram's Macro's by example, see http://www.ccs.neu.edu/home/dorai/.
  • src/elf/md5.scm Renu Bostwick's MD5 signature code (uses NullOutputStream).
  • src/elf/paste.scm Example of cut, copy and paste from
