To install a JScheme-webapp you
  1. Download and install a servlet container such as tomcat, jetty, or resin, then
  2. Download jscheme-webapp.zip and unzip it in the webapps folder of the servlet container
  3. Restart the servlet container (or use one of the servlet administration tools to start the jscheme servlet...)


You may also want to configure the webapp by modifying the WEB-INF/web.xml file.
