An Interval Arithmetic Evaluator Demo in Scheme

The source code for this applet is in the files
  • iaevaldemo.scm -- the GUI
  • iaeval.scm -- the Interval Arithmetic Evaluator
  • opparse.scm -- an Operator Precedence parser
In addition, this uses the ia_math package of Java classes which implement interval arithmetic versions of the standard Math operators and functions. This package will soon be available at interval.sourceforge.net.
