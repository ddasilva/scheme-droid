We have several interval arithmetic applications written in JScheme.

  1. IA-Solver is a solver for non-linear constraints. It is based on the Interval Arithmetic constraint solving package ia_math from interval.sourceforge.net
  2. IA-Eval is a simple Interval Arithmetic Evaluator
