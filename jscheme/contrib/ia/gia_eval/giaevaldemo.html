A General Interval Arithmetic Evaluator Demo in Scheme

The source code for this applet is in the files
  • ereal.scm -- an implementation of extended reals
  • gia.scm -- an implementation of interval arithmetic
  • opparse.scm -- an Operator Precedence parser
  • giaeval.scm -- the General Interval Arithmetic Evaluator
  • giaevaldemo.scm -- the GUI
